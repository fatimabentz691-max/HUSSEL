# 更新记录

本项目遵循语义化版本思路，当前处于快速迭代阶段。

## 0.2.0 - 2026-07-14

### 新增

- Token Manager 全新产品名称与黑白应用视觉；
- Codex 专属本地日志仪表盘和用户自定义 5 小时/7 天预算；
- 全部模型、Codex 和单模型独立仪表盘；
- 7/30 天多 AI 彩色 Token 堆叠柱状图；
- DeepSeek 官方人民币余额同步与余额历史；
- OpenAI、Anthropic、Gemini、DeepSeek 代理 usage 解析；
- 多账户 DPAPI 密钥持久化、单条删除和批量清空；
- 一键启动全部账户本地代理；
- 可拖动、缩放、折叠和切换模型的置顶悬浮窗；
- 悬浮窗模块勾选、排序、网格/列表和迷你模式；
- CSV 账单导出与 `.tmbak` 加密迁移；
- CC Switch 本地连接检测；
- 18 个平台官方来源 Logo，本地离线打包显示。

### 准确性说明

- Codex 剩余额度为本地预算估算，不是 OpenAI 官方会员余额；
- 未验证厂商官方账单接口时只显示代理统计或能力限制；
- 不生成模拟 Token、余额或消费记录。
