# Token Manager 0.2.0

Token Manager 是面向 Windows 10/11 开发者的本地 AI Token 用量管理工具。本版本提供 Codex 专属仪表盘、全部模型总览、可配置悬浮窗、账户与模型管理、本地代理计量、消费趋势与 CSV 账单导出。

## 本版重点

- 独立 Codex 仪表盘：5 小时与 7 天滚动窗口、本地日志用量估算、刷新倒计时与阈值提醒。
- 全部模型仪表盘：按供应商区分 Token、请求次数、缓存命中、消费金额和余额。
- 可拖动、置顶、可缩放的模块化悬浮窗，支持模型切换、迷你模式和布局持久化。
- DeepSeek 及 OpenAI 兼容代理实时计量：只记录模型、Token、时间、状态与成本，不保存请求或响应正文。
- 多账户配置与本地加密密钥持久化，支持单条删除和批量清空。
- 中文 iOS 浅色视觉、黑白应用图标、供应商官方品牌素材适配。

## 数据口径说明

Codex Plus 不提供面向本工具的官方余额接口，因此 Codex 额度为本地日志观测与用户自定义 5 小时/7 天额度上限计算的参考值，不会伪装成官方余额。API 平台仅在供应商提供且账户具备相应权限时显示官方余额；否则明确标记为本地代理、账单导入或估算数据。
