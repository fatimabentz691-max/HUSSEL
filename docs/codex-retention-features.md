# Codex 留存功能使用说明

## 数据口径

Codex 专属页面只读取本机 Codex 状态库与日志中的时间、模型、Token、turn 标识和可用运行参数，不读取提示词或代码正文。5 小时/7 天百分比、开发时长、任务数量和人民币金额均为本地观测推算，不是 OpenAI 官方账户余额。

## 开发能力换算

- 完整项目生成时长：以最近日志的活跃小时 Token 速率为基础，对高消耗生成场景应用修正系数。
- 局部调试时长：使用同一历史速率，对低消耗调试场景应用修正系数。
- 任务数量：默认以 12,000 Token/小脚本、24,000 Token/页面重构、8,000 Token/Bug 修复换算。
- 消耗分类：日志包含任务标记时优先使用标记；没有标记时，按单次 Token 规模启发式分为全新生成、修改调试、问答解释。

## 项目标签与复盘

在“Codex 专属 → 高消耗会话复盘”中为每条 turn 输入项目标签，可使用预置标签或自定义文字。标签、筛选状态和统计均保存在本机。导出按钮生成 Excel 可直接打开的 `.xls` 消耗账单。

## 重置券

重置券由用户手动登记数量和到期日期。点击“记录使用”会保存使用次数，并按当时的 7 天已用 Token 记录节省额度。到期前三天以及 5 小时额度低于 20% 时会提示。

## 价格目录

内置目录位于 `src/features/retention.ts`，包含版本号、生效日期、价格来源状态和每百万 Token 的输入/输出/缓存价格。只有“已核验”条目可作为精确计价来源；“内置参考”只用于无官方账单接口时的预算比较。更新目录不需要修改统计核心。

## 悬浮窗

设置页可启用“剩余开发预估时长”和“可执行任务数量”，并与其他模块一起拖拽排序、设置迷你模式。20%/10% 预警会显示红色脉冲文字；系统开启“减少动态效果”后自动停止闪烁。

## 加密迁移

在“设置 → 本地加密迁移”输入至少 8 位迁移密码。导出的 `.tmbak` 使用 PBKDF2-SHA256 派生密钥和 ChaCha20-Poly1305 加密，包含账户密钥、历史用量、预算、仪表盘、项目标签与悬浮窗布局。导入到新电脑时，API Key 会重新使用该电脑当前 Windows 用户的 DPAPI 加密。

迁移密码不会保存，遗失后无法恢复备份。
